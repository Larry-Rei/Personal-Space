《Flappy Bird 重构版》

项目描述：一款经典的 2D 侧翻滚无尽飞行游戏，核心亮点在于引入 xLua 框架对原生 C# 玩法进行了全面重构，实现了“C# 底层稳定承载 + Lua 业务热更”的混合编程架构。

Unity 2DC# / Lua 混合编程xLua 热更框架DOTween对象池优化

⚙️ 核心架构与逻辑拆解

  • xLua 混合架构与生命周期接管：设计并实现 WidgetLuaBehaviour 通用承载组件。通过 Inspector 依赖注入，完成 C# 引用类型向 Lua 虚拟机的零开销映射。接管并转发 Unity 原生生命周期，为每个挂载对象开辟独立的 LuaTable 作用域，彻底解耦底层框架与上层玩法。
  • 核心 3C 与视觉表现的 Lua 化重构：摒弃 Unity 原生物理组件，在 Lua 端运用纯代码计算欧拉积分，精准模拟重力下坠与点击上抛的物理反馈。引入 DOTween 替代原生协程，在 Lua 侧处理序列帧动画与定时器，兼顾了开发的灵活性与平滑的视觉表现。
  • Lua 端对象池与性能防抖：针对无尽生成的管道障碍物，在 Lua 侧独立构建了轻量级的数组缓存池 (Object Pool)。通过状态切换接管高频的 Instantiate/Destroy 操作，大幅规避了跨语言边界下产生的 GC 峰值，保障了跨设备运行的帧率稳定性。
